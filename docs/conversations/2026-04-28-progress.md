### 4. ActionFormer config (class-agnostic)

**ファイル**: `codes/ActionFormer/configs/tal_motion_vit_b.yaml`

**主要設定**:
```yaml
dataset_name: tal_motion
dataset:
  num_classes: 1              # class-agnostic
  input_dim: 768              # VideoMAE v2-B
  feat_stride: 2              # 0.2 秒/step
  num_frames: 16
  default_fps: 10.0
  max_seq_len: 192            # 30s 動画に対応
  feat_folder: data/features/30s_mae_b_16_2
  annot_dir: $ANNOT_ROOT_DIR
  split_list_dir: data/splits/v1       # 後述
  aux_vocab_file: data/aux_stats/v1/aux_vocab.json  # 後述

model:
  backbone_arch: [2, 2, 4]
  regression_range: [[0, 2], [2, 4], [4, 8], [8, 16], [16, 10000]]
  embd_dim: 384
  n_head: 4
```

**補助分類ヘッド** (Phase 3 キャプション向け):
- `body_part` (30+ クラス)
- `action_type` (31 クラス)
- `grip_or_contact` (分類)
- `speed_or_force` (分類)
- `posture_change` (分類)

## ⏳ GPU ノード側で実行するタスク

### split 生成（train/val/test）

```bash
export UV_CACHE_DIR=/lustre/work/mt/.uv-cache
export HF_HOME=/lustre/work/mt/.cache/huggingface
export ANNOT_ROOT_DIR=/raid/containers/enroot/data/b100/ft_fact_videos/annot/30s_chunks_action_31detail_2

cd /lustre/work/mt/okamura/tal-annotation-labeling

uv run --no-sync python codes/VideoMAEv2/tools/sample_subset.py \
  --annot-dir "$ANNOT_ROOT_DIR" \
  --out-dir data/splits/v1 \
  --val-frac 0.05 --test-frac 0.05 --seed 0
```

**出力**:
- `data/splits/v1/train.txt` (1 行 1 json ファイル名)
- `data/splits/v1/val.txt`
- `data/splits/v1/test.txt`
- `data/splits/v1/split.json` (統計)

追記：実行済み

### aux vocab 生成（5 補助フィールド）


**出力**:
